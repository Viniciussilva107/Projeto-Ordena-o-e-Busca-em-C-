# Projeto Ordencao e Busca em C
Projeto em C de ordenação e busca usando memória secundaria, feito para a disciplina de Analise e Estrutura de Dados 2. 
Foi feito um projeto para realizar buscas e ordenações em uma clinica de adoção de animais. 
O projeto usa estruturas de ordenação e busca para organizar os animais e adotantes por ID, tambem faz o controle de adoção.
No trabalho 1 foi usado Busca Binaria e Bubble Sort para ordenar.
No trabalho 2 foi usado metodo de ordenação externa.
No trabalho 3 foi todo em volta de tabela Hash.
O projeto foi todo feito em memoria Secundaria. Ele conta com um log em txt que deixa todo os historico de modificações registrado, assim como o tempo de execução de cada metodo, mostrando qual é mais veloz e qual usar em cada ocasião.
